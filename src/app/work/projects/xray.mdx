## Overview

Welcome to **pneumonia-detection-xray**!

This project implements an end-to-end deep learning workflow for detecting pneumonia from pediatric chest X-ray images. By utilizing transfer learning with a pre-trained **ResNet-18** architecture, the model efficiently extracts high-level spatial features and classifies chest X-rays into **Normal** or **Pneumonia** categories. The system is deployed as an interactive web app for quick inference.

### Key Features
- **Transfer Learning Workflow**: Leverages a pre-trained **ResNet-18** model, freezing the convolutional feature extractor to train a custom fully-connected classifier head for binary classification.
- **Robust Data Pipeline**: Processes chest X-ray images with specialized PyTorch transformations (resizing, tensor conversion, and ImageNet normalization) to prepare them for neural network inference.
- **Interactive Streamlit Web App**: An intuitive user interface where clinicians or developers can upload chest X-ray images (JPEG/PNG) and get real-time visual output along with the predicted class.
- **Automated Validation & Evaluation**: Features training and evaluation scripts that track validation accuracy and save the best-performing model checkpoint (`pneumonia_model.pth`) based on validation set performance.
