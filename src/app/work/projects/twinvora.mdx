## Overview

Welcome to **Twinvora**!

Twinvora is an Adaptive Clinical Intelligence Platform designed to provide doctors with actionable, predictive insights using "Patient Digital Twins" and Ambient Intelligence. It is built to be HIPAA-compliant, FDA SaMD (Software as a Medical Device) aligned, and capable of running in low-resource environments.

This project was developed by Dwaipayan Dasgupta.
